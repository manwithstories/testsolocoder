# 太空射击游戏 (Space Shooter)

一个使用 Python 和 Pygame 开发的太空射击游戏，具有完整的游戏系统。

## 功能特性

### 核心玩法
- **玩家控制**：使用方向键或 A/D 键左右移动飞船，空格键发射子弹
- **敌人系统**：多种移动模式的敌人（直线、Z字形、正弦波）
- **波次系统**：每波敌人数量和速度递增，波次间有休息时间

### 道具系统
- **护盾 (S)**：激活后免疫伤害一段时间
- **快速射击 (F)**：提升射击速度
- **生命恢复 (+)**：恢复一点生命值
- **散射 (M)**：同时发射三发子弹

### 商店系统
每 5 波后开启商店，可以用积分升级：
- 子弹威力：500 积分
- 移动速度：400 积分
- 射击速度：600 积分
- 恢复生命：300 积分

### Boss 战
- 每 5 波出现一个 Boss
- Boss 有血条显示
- 三种攻击模式：散射、连射、三连发

## 安装和运行

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行游戏
```bash
python main.py
```

## 操作说明

| 按键 | 功能 |
|------|------|
| ← / A | 向左移动 |
| → / D | 向右移动 |
| 空格 | 射击 |
| ↑ / ↓ | 菜单选择 |
| 回车 | 确认选择 |

## 游戏状态

- **主菜单**：开始游戏或退出
- **波次休息**：显示即将开始的波次，倒计时 3 秒
- **战斗中**：正常游戏
- **商店**：购买升级
- **游戏结束**：显示最终分数，可以重新开始或返回主菜单

## 技术实现

- 使用 Pygame 进行图形渲染和输入处理
- 模块化设计，各系统独立
- 状态机管理游戏流程
- 程序化生成音效（无需外部音频文件）
- 星空背景动画
